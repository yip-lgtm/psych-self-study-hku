# PSYC2103 — Foundations of Neuroscience II

## Course Info (from HKU Catalog)

- **Code:** PSYC2103
- **Title:** Foundations of neuroscience II
- **Credits:** 6
- **HKU Catalog Description:** This course complements the first course in this series, Foundations of Neuroscience I, covering the fundamental principles of neuroscience. This second course will familiarise students with principles related to chemical control of the brain (e.g., homeostasis, hypothalamic control, and relevance to behaviour), the autonomic nervous system, and diffuse modulatory systems (e.g., noradrenergic, serotonergic, dopaminergic, and cholinergic regulation). This course will culminate with a series of introductory programming modules to prepare students for advanced research at the undergraduate and graduate levels. Priority will be given to students majoring/minoring in neuroscience.
- **Prerequisites:** PSYC1001 and PSYC2101
- **Assessment:** 100% coursework
- **Notes:** Programming modules; deeper neurochem focus

## Learning Objectives

By the end of this course, you should be able to:

1. Describe chemical control of the brain
2. Explain the autonomic nervous system
3. Discuss diffuse modulatory systems
4. Apply programming to neuroscience data
5. Conduct basic computational analyses
6. Prepare for advanced research

## Topic Outline

### Topic 1: Homeostasis (3 hours)
- Definition
- Hypothalamic integration
- Temperature, hunger, thirst, fluid balance
- Set-point theory

### Topic 2: Hypothalamic Control (4 hours)
- Anatomy of hypothalamus
- Nuclei and functions
- Autonomic integration
- Endocrine integration

### Topic 3: Autonomic Nervous System (4 hours)
- Sympathetic
- Parasympathetic
- Enteric
- Fight-or-flight
- Rest-and-digest

### Topic 4: Diffuse Modulatory Systems I (4 hours)
- Noradrenergic (locus coeruleus)
- Serotonergic (raphe nuclei)
- Arousal and mood

### Topic 5: Diffuse Modulatory Systems II (4 hours)
- Dopaminergic (VTA, SNc)
- Cholinergic (basal forebrain, brainstem)
- Reward and movement

### Topic 6: Stress and the HPA Axis (3 hours)
- Cortisol
- Allostasis vs. homeostasis
- Chronic stress effects

### Topic 7: Reproductive Hormones (3 hours)
- HPG axis
- Sexual differentiation
- Organizational vs. activational effects
- Oxytocin, vasopressin

### Topic 8: Sleep and Circadian Rhythms (3 hours)
- Circadian rhythms
- Sleep stages
- Sleep-wake neurochemistry
- Disorders

### Topic 9: Introduction to Programming (6 hours)
- Python basics
- NumPy, pandas
- Data visualization (matplotlib)
- Working with neural data

### Topic 10: Computational Neuroscience Methods (6 hours)
- Spike train analysis
- Signal processing basics
- Modeling approaches
- Reproducible workflows

## Key Concepts Glossary

| Term | Definition |
|---|---|
| Homeostasis | Maintenance of internal balance |
| Allostasis | Adaptive change to maintain stability |
| Hypothalamus | Master regulatory structure |
| Autonomic NS | Involuntary functions |
| Sympathetic | Fight-or-flight |
| Parasympathetic | Rest-and-digest |
| HPA axis | Stress response system |
| Cortisol | Stress hormone |
| Circadian | ~24-hour cycle |
| Locus coeruleus | Noradrenergic nucleus |
| Raphe nuclei | Serotonergic nuclei |
| VTA | Ventral tegmental area (dopamine) |
