# PSYC2002 — Psychological Testing and Measurement

## Course Info (from HKU Catalog)

- **Code:** PSYC2002
- **Title:** Psychological testing and measurement
- **Credits:** 6
- **HKU Catalog Description:** This course surveys the major concepts and techniques in the field of psychometrics, and provides students with some hands-on experience with commonly used tests. Topics covered include: the context of testing and measurement; the testing process; test standardization; reliability and validity; intelligence and its appraisal; personality assessment; special domain testing; occupational applications; large-scale measurements; ethics and prospects.
- **Prerequisites:** PSYC1001
- **Assessment:** 100% coursework
- **Notes:** Practical with common tests; psychometric foundation

## Learning Objectives

By the end of this course, you should be able to:

1. Describe the major concepts in psychometrics
2. Distinguish types of reliability and validity
3. Administer and interpret common psychological tests
4. Evaluate intelligence theories and assessment methods
5. Describe personality assessment approaches
6. Apply testing in occupational settings
7. Discuss ethical issues in psychological testing

## Topic Outline

### Topic 1: Context of Testing (3 hours)
- History of psychological testing (Binet, Galton, Wechsler)
- Testing in modern society
- Cultural and ethical considerations

### Topic 2: Test Construction Process (4 hours)
- Test planning
- Item writing
- Pilot testing
- Standardization
- Norms and reference groups

### Topic 3: Reliability (5 hours)
- Test-retest reliability
- Internal consistency (Cronbach's α, McDonald's ω)
- Inter-rater reliability
- Parallel forms
- Factors affecting reliability

### Topic 4: Validity (5 hours)
- Content validity
- Criterion validity (concurrent, predictive)
- Construct validity
- Face validity
- Factor analysis (EFA, CFA)

### Topic 5: Intelligence (6 hours)
- Spearman's g
- Gardner's multiple intelligences
- Sternberg's triarchic theory
- Wechsler scales (WAIS, WISC)
- IQ controversies and heritability

### Topic 6: Personality Assessment (5 hours)
- Self-report inventories (MMPI, NEO-PI-R, 16PF)
- Projective tests (Rorschach, TAT) — controversial
- Behavioral assessment
- Implicit measures (IAT)

### Topic 7: Special Domain Testing (4 hours)
- Clinical assessment
- Neuropsychological testing
- Achievement testing
- Aptitude testing

### Topic 8: Occupational Applications (4 hours)
- Job analysis
- Personnel selection
- Performance appraisal
- 360-degree feedback
- Assessment centers

### Topic 9: Large-Scale Assessments (3 hours)
- PISA, TIMSS
- PIAAC
- Standardized testing programs

### Topic 10: Ethics and Future (3 hours)
- Test fairness and bias
- AI in testing
- Privacy concerns
- Future of psychometrics

## Key Concepts Glossary

| Term | Definition |
|---|---|
| Psychometrics | Theory and method of psychological measurement |
| Reliability | Consistency of measurement |
| Validity | Accuracy of measurement (does it measure what it claims?) |
| Standardization | Uniform administration, scoring, and interpretation |
| Norms | Reference data for comparing individual scores |
| IQ | Intelligence quotient, originally (mental age / chronological age) × 100 |
| Factor analysis | Statistical technique identifying clusters of related items |
| Item response theory | Modern psychometric framework modeling item-level performance |

## Self-Check Questions

1. Distinguish reliability from validity. Can a test be valid but unreliable?
2. Compare Spearman's g and Gardner's multiple intelligences.
3. What is Cronbach's alpha? What value is considered acceptable?
4. Discuss the ethical issues in using AI-based personality assessment for hiring.
5. What is the difference between EFA and CFA? When would you use each?
