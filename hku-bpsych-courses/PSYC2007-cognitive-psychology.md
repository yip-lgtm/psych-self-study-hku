# PSYC2007 — Cognitive Psychology

## Course Info (from HKU Catalog)

- **Code:** PSYC2007
- **Title:** Cognitive psychology
- **Credits:** 6
- **HKU Catalog Description:** This course covers how humans process information from the environment. Topics include various aspects of perception, attention, memory, imagery, language and decision-making. Students will learn from attending lectures and active participation during tutorials. Students will also conduct experiments about cognitive functioning and learn to critically evaluate existing studies in the research literature and to write research reports on experimental findings.
- **Prerequisites:** PSYC1001 and PSYC1004
- **Assessment:** 100% coursework
- **Notes:** Lab-based; conduct experiments; write research reports

## Learning Objectives

By the end of this course, you should be able to:

1. Describe major cognitive processes (perception, attention, memory, language, decision-making)
2. Conduct cognitive psychology experiments
3. Analyze and interpret experimental data
4. Critically evaluate cognitive psychology research
5. Write a research report in APA format
6. Apply cognitive psychology to real-world problems

## Topic Outline

### Topic 1: Introduction to Cognitive Psychology (2 hours)
- History (cognitive revolution)
- Information-processing approach
- Levels of analysis (Marr)

### Topic 2: Perception (5 hours)
- Bottom-up vs. top-down
- Gestalt principles
- Feature detection
- Illusions

### Topic 3: Attention (5 hours)
- Selective attention
- Divided attention
- Sustained attention
- Inattentional blindness
- Change blindness

### Topic 4: Memory I — Sensory and Short-Term (4 hours)
- Sensory memory
- Short-term memory capacity
- Working memory model (Baddeley)
- Encoding modalities

### Topic 5: Memory II — Long-Term (5 hours)
- Types of LTM (episodic, semantic, procedural)
- Encoding and retrieval
- Forgetting (decay, interference)
- Memory distortion
- Eyewitness testimony

### Topic 6: Mental Imagery (3 hours)
- Image properties
- Mental rotation
- Cognitive maps
- Debate: depictional vs. propositional

### Topic 7: Language (5 hours)
- Levels of language analysis
- Speech perception
- Comprehension
- Production
- Bilingualism

### Topic 8: Decision Making (5 hours)
- Dual-process theory
- Heuristics and biases
- Prospect theory
- Bounded rationality

### Topic 9: Experimental Methods in Cognition (4 hours)
- Reaction time
- Accuracy measures
- Eye tracking
- Signal detection theory

### Topic 10: Lab Reports and Critical Evaluation (4 hours)
- APA format
- Literature review
- Method, Results, Discussion
- Critical thinking about claims

## Key Concepts Glossary

| Term | Definition |
|---|---|
| Cognitive psychology | Scientific study of mental processes |
| Information processing | Input → processing → output model |
| Working memory | Active manipulation of information |
| Levels of processing | Depth of processing predicts retention |
| Heuristic | Mental shortcut |
| Bias | Systematic deviation from rational judgment |
| Reaction time | Time to respond to a stimulus |
| Signal detection theory | Framework for separating signal from noise |

## Self-Check Questions

1. Compare sensory memory, short-term memory, and long-term memory.
2. Describe Baddeley's model of working memory.
3. What is the difference between recall and recognition? When is each better?
4. Describe the Stroop effect. What does it tell us about cognition?
5. How would you design an experiment to test inattentional blindness?
