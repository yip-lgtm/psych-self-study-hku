# PSYC2022 — Biological Psychology

## Course Info (from HKU Catalog)

- **Code:** PSYC2022
- **Title:** Biological psychology
- **Credits:** 6
- **HKU Catalog Description:** This course provides an introduction to biological aspects of behaviour. The topics include: biological bases of behaviour, development, learning, memory, and abnormal psychology; the nervous system; processes of brain maturation; psychophysiology.
- **Prerequisites:** PSYC1001
- **Assessment:** 100% coursework
- **Notes:** Foundation for higher-level neuroscience courses (PSYC2101+)

## Learning Objectives

By the end of this course, you should be able to:

1. Describe the structure and function of the nervous system
2. Explain how neurons communicate
3. Identify major neurotransmitter systems
4. Discuss biological bases of behavior, learning, memory
5. Describe psychophysiology methods
6. Apply biological psychology to abnormal behavior

## Topic Outline

### Topic 1: Introduction to Biological Psychology (3 hours)
- History (Descartes, Broca, Sperry)
- Reductionism and holism
- Mind-body problem

### Topic 2: Nervous System Overview (4 hours)
- CNS vs. PNS
- Brain divisions (forebrain, midbrain, hindbrain)
- Spinal cord

### Topic 3: The Neuron (5 hours)
- Structure
- Action potential
- Synaptic transmission
- Glial cells

### Topic 4: Neurotransmitters (4 hours)
- Acetylcholine, dopamine, serotonin
- Norepinephrine, GABA, glutamate
- Endorphins
- Drug effects on neurotransmitters

### Topic 5: Brain Anatomy (5 hours)
- Cerebral cortex (4 lobes)
- Subcortical structures
- Limbic system
- Brainstem and cerebellum

### Topic 6: Brain Development (3 hours)
- Prenatal development
- Neurogenesis
- Synaptic pruning
- Critical periods

### Topic 7: Biological Bases of Learning (3 hours)
- Hebbian learning
- Long-term potentiation
- Role of hippocampus
- Classical conditioning neuroscience

### Topic 8: Biological Bases of Memory (3 hours)
- Types of memory
- Hippocampus and declarative memory
- Amygdala and emotional memory
- Cerebellum and procedural memory

### Topic 9: Psychophysiology (4 hours)
- EEG
- fMRI, PET
- Heart rate, skin conductance
- Eye tracking
- Neuroimaging methods

### Topic 10: Biological Bases of Abnormal Psychology (4 hours)
- Schizophrenia (dopamine hypothesis)
- Depression (monoamine hypothesis)
- Anxiety disorders
- Substance use disorders
