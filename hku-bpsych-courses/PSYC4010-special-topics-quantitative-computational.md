# PSYC4010 — Special Topics in Psychology IV (Quantitative and Computational Methods)

## Course Info (from HKU Catalog)

- **Code:** PSYC4010
- **Title:** Special topics in psychology IV (quantitative and computational methods)
- **Credits:** 6
- **HKU Catalog Description:** This advanced seminar course focuses on a specific topic at the cutting edge of psychology research. Topic varies by offering. Possible topics include psychometrics, quantitative psychology, computational modeling, machine learning in psychology, Bayesian data analysis, network analysis, and other emerging methods. May involve substantial reading, original research, and presentation. Restricted to final-year BPsych students.
- **Prerequisites:** PSYC1004, PSYC2060, PSYC2061
- **Assessment:** 100% coursework
- **Notes:** 4000-level honours seminar; methods-focused

## Learning Objectives

By the end of this course, you should be able to:

1. Engage deeply with advanced quantitative methods
2. Apply psychometrics to scale development
3. Conduct Bayesian data analysis
4. Apply machine learning to psychological data
5. Conduct network analysis
6. Apply computational modeling
7. Critically evaluate methodological choices

## Topic Outline (Example: Quantitative and Computational Methods)

### Topic 1: Course Introduction (3 hours)
- Topic scope
- Methodological focus
- Student presentations
- Research project scope

### Topic 2: Advanced Psychometrics (5 hours)
- Item response theory (IRT)
- Differential item functioning (DIF)
- Rasch modeling
- Computerized adaptive testing

### Topic 3: Multilevel Modeling Advanced (5 hours)
- Cross-classified models
- Three-level models
- Longitudinal MLM
- Non-linear MLM

### Topic 4: Structural Equation Modeling Advanced (5 hours)
- Multi-group SEM
- Longitudinal SEM
- Latent growth curve models
- Measurement invariance testing

### Topic 5: Bayesian Data Analysis (5 hours)
- Bayes' theorem
- Priors and posteriors
- MCMC
- Software (Stan, brms, JASP)

### Topic 6: Machine Learning in Psychology (5 hours)
- Supervised vs. unsupervised
- Random forests, SVM
- Neural networks
- Cross-validation
- Application examples

### Topic 7: Network Analysis (4 hours)
- Psychological networks
- Estimation
- Stability
- Interpretation

### Topic 8: Computational Modeling (4 hours)
- Reinforcement learning models
- Drift diffusion
- Decision-making models
- Model comparison

### Topic 9: Causal Inference (3 hours)
- Counterfactuals
- Directed acyclic graphs
- Instrumental variables
- Natural experiments

### Topic 10: Reproducibility and Open Science (3 hours)
- Pre-registration
- Open data
- Open materials
- Registered reports
- HK and international initiatives

### Topic 11: Research Project (5 hours)
- Apply methods to existing dataset
- Conference-style presentation
- Final paper

## Key Concepts Glossary

| Term | Definition |
|---|---|
| IRT | Item response theory |
| DIF | Differential item functioning |
| Bayesian | Updating beliefs with data |
| MCMC | Markov chain Monte Carlo |
| Machine learning | Algorithms learning from data |
| Cross-validation | Testing on held-out data |
| Network analysis | Modeling as a network |
| Drift diffusion | Sequential sampling model |
| DAG | Directed acyclic graph |
| Pre-registration | Publicly registering hypotheses |
| Open science | Transparent research practices |
| Measurement invariance | Test equivalence across groups |

## Self-Check Questions

1. Fit a 2-parameter IRT model and interpret.
2. Conduct Bayesian regression using brms.
3. Apply a random forest to a psychological dataset.
4. Estimate a psychological network and interpret.
5. Discuss the role of pre-registration in modern psychology.
