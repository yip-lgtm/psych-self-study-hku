# PSYC1004 — Introduction to Quantitative Methods in Psychology

## Course Info (from HKU Catalog)

- **Code:** PSYC1004
- **Title:** Introduction to quantitative methods in psychology
- **Credits:** 6
- **HKU Catalog Description:** This course adopts a practical approach to teaching the analytical aspects of research in psychology. The course is designed to provide students with the basic background in research design and data analysis, covering the logic of statistical reasoning and inference as well as the key concepts involved. Priority will be given to students who plan to major in psychology.
- **Prerequisites:** None
- **Assessment:** 100% coursework
- **Notes:** Foundation for all higher-level quant methods; prereq for PSYC2007, PSYC2060, PSYC2061

## Learning Objectives

By the end of this course, you should be able to:

1. Describe the logic of statistical reasoning and inference
2. Choose appropriate research designs for psychological questions
3. Compute and interpret descriptive statistics
4. Conduct basic inferential tests (t-test, correlation, chi-square)
5. Interpret statistical output (p-values, effect sizes, confidence intervals)
6. Use statistical software (JASP, R, or SPSS)
7. Critically evaluate quantitative claims in research

## Topic Outline

### Topic 1: Introduction to Research Design (4 hours)
- Scientific method
- Variables (IV, DV, confound, moderator, mediator)
- Experimental vs. correlational designs
- Sampling methods

### Topic 2: Measurement (3 hours)
- Levels of measurement (nominal, ordinal, interval, ratio)
- Reliability and validity
- Operationalization

### Topic 3: Descriptive Statistics (5 hours)
- Central tendency (mean, median, mode)
- Variability (range, SD, IQR)
- Distributions and the normal curve
- Z-scores

### Topic 4: Probability and Sampling (4 hours)
- Probability basics
- Sampling distributions
- Standard error
- Central Limit Theorem

### Topic 5: Hypothesis Testing Logic (4 hours)
- Null and alternative hypotheses
- Type I and Type II errors
- Alpha and beta
- p-values
- Statistical power

### Topic 6: t-Tests (5 hours)
- Independent samples t-test
- Paired samples t-test
- One-sample t-test
- Effect size: Cohen's d

### Topic 7: Correlation (4 hours)
- Pearson's r
- Significance testing
- Causation vs. correlation
- Visualization (scatterplot)

### Topic 8: Chi-Square (3 hours)
- Goodness of fit
- Test of independence
- Effect size: Phi, Cramér's V

### Topic 9: ANOVA Intro (4 hours)
- One-way ANOVA
- F-distribution
- Post-hoc tests

### Topic 10: Statistical Software (4 hours)
- JASP (recommended, free)
- R or SPSS
- Hands-on analysis

## Key Concepts Glossary

| Term | Definition |
|---|---|
| p-value | Probability of obtaining results at least as extreme as observed, assuming null is true |
| Effect size | Standardized magnitude of an effect (e.g., Cohen's d) |
| Confidence interval | Range likely to contain the true population parameter (typically 95%) |
| Power | Probability of correctly rejecting a false null (typically 0.80) |
| Standard error | SD of the sampling distribution |
| Effect size | Standardized measure of effect magnitude |
| Type I error | False positive (rejecting true null) |
| Type II error | False negative (failing to reject false null) |

## Self-Check Questions

1. Explain the difference between a Type I and Type II error. Which is more serious in your capstone context?
2. A study reports t(48) = 2.34, p = .024, d = 0.65. Interpret fully.
3. Why does correlation not imply causation? Give a 3-variable example.
4. Calculate the mean, median, mode, and SD for: 3, 5, 7, 7, 9, 11, 13.
5. What is statistical power? What 3 things can you do to increase it?
