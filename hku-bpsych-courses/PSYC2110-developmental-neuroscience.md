# PSYC2110 — Developmental Neuroscience

## Course Info (from HKU Catalog)

- **Code:** PSYC2110
- **Title:** Developmental neuroscience
- **Credits:** 6
- **HKU Catalog Description:** Developmental neuroscience is an interdisciplinary research topic that integrates neuroscience, cognitive science and developmental science. This course aims to uncover the brain and neural mechanisms that underlie social, affective and cognitive development across the life span. Specific topics will include the introduction of theories and methods in developmental neuroscience, neuroplasticity, neural mechanisms that underlie the development of attention and perception processes, motor learning, memory, cognitive control, social-emotional processes. This course will examine these processes at different developmental stages, including infants, toddlers, adolescence and ageing population. This course will also cover the neural mechanisms underlying atypical development such as the Autism Spectrum Disorder (ASD).
- **Prerequisites:** PSYC2101
- **Assessment:** 100% coursework
- **Notes:** Integrates developmental + cognitive + neuroscience

## Learning Objectives

By the end of this course, you should be able to:

1. Describe theories and methods in developmental neuroscience
2. Discuss neuroplasticity across the lifespan
3. Explain neural mechanisms of attention, perception, memory development
4. Discuss social-emotional brain development
5. Describe neural mechanisms of motor learning
6. Discuss atypical development (e.g., ASD)

## Topic Outline

### Topic 1: Introduction to Developmental Neuroscience (3 hours)
- Historical roots
- Methods (EEG/ERP, fNIRS, MRI in infants)
- Ethics of infant/child research

### Topic 2: Brain Development (5 hours)
- Prenatal brain development
- Neurogenesis, migration, differentiation
- Synaptogenesis and pruning
- Myelination
- Sensitive periods

### Topic 3: Neuroplasticity (4 hours)
- Experience-dependent plasticity
- Critical periods
- Adult plasticity
- Mechanisms (LTP, structural changes)

### Topic 4: Attention and Perception Development (4 hours)
- Visual system development
- Face processing
- Attention networks
- ERP markers of cognitive development

### Topic 5: Motor Development (3 hours)
- Motor cortex development
- Corticospinal tract
- Motor learning
- Dyspraxia

### Topic 6: Memory Development (4 hours)
- Hippocampal development
- Infant memory
- Episodic memory emergence
- Source memory
- Childhood amnesia

### Topic 7: Cognitive Control Development (4 hours)
- Prefrontal cortex maturation
- Executive function
- Inhibitory control
- Working memory development
- Adolescent brain

### Topic 8: Social-Emotional Development (4 hours)
- Attachment and brain
- Face processing and social brain
- Theory of mind and mentalizing
- Emotion regulation
- Adolescence and peer influence

### Topic 9: Language Development (3 hours)
- Language areas development
- Critical period for language
- Bilingual brain
- Reading and dyslexia

### Topic 10: Aging Brain (3 hours)
- Normal cognitive aging
- Neuroplasticity in older age
- Dementia

### Topic 11: Atypical Development (5 hours)
- Autism Spectrum Disorder
- ADHD
- Intellectual disability
- Specific language impairment
- Early intervention and brain plasticity

## Key Concepts Glossary

| Term | Definition |
|---|---|
| Developmental neuroscience | Brain and behavior across lifespan |
| Neuroplasticity | Brain change with experience |
| Synaptogenesis | Formation of synapses |
| Pruning | Elimination of synapses |
| Sensitive period | Optimal time for development |
| Myelination | Insulation of axons |
| Prefrontal cortex | Executive function hub |
| ERP | Event-related potential (EEG measure) |
| fNIRS | Functional near-infrared spectroscopy |
| Childhood amnesia | Inability to recall early life events |
| Theory of mind | Understanding others' mental states |
| Mentalizing | Reasoning about others' minds |
