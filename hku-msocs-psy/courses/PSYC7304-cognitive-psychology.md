# PSYC7304 — Cognitive Psychology

## Course Info

- **Code:** PSYC7304
- **Title:** Cognitive psychology
- **MSocSc(Psy) Compulsory:** Yes (one of 8)
- **BPsych Equiv:** PSYC2007 / PSYC2051
- **Self-Study Month:** 2
- **Estimated Study Hours:** 40–50 hours

## Learning Objectives

By the end of this course, you should be able to:

1. Explain the information-processing approach to cognition
2. Describe the structure and limits of attention
3. Distinguish types of memory (sensory, short-term, working, long-term)
4. Explain how knowledge is represented in long-term memory
5. Describe language processing (production and comprehension)
6. Apply cognitive psychology to decision making and problem solving
7. Critically evaluate claims about cognitive enhancement
8. Discuss the relationship between cognition and the brain

## Topic Outline

### Topic 1: Introduction to Cognitive Psychology (2 hours)
- Mind as information processor
- History (behaviorism → cognitive revolution)
- Computer metaphor: input → processing → output
- Levels of analysis (Marr's computational, algorithmic, implementational)

### Topic 2: Perception I — Bottom-Up Processing (5 hours)
- Sensation vs. perception
- Gestalt principles (proximity, similarity, closure, continuity)
- Feature detection (Hubel & Wiesel)
- Top-down vs. bottom-up processing
- Illusions (Müller-Lyer, Ponzo)

### Topic 3: Perception II — Attention (5 hours)
- Selective attention (cocktail party effect)
- Broadbent's filter model
- Treisman's attenuation model
- Late selection (Deutsch & Deutsch)
- Visual search (feature vs. conjunction)
- Attention networks (Posner): alerting, orienting, executive
- Inattentional blindness (Simons & Chabris, 1999)

### Topic 4: Short-Term and Working Memory (5 hours)
- Atkinson-Shiffrin multi-store model
- Baddeley's working memory model (phonological loop, visuospatial sketchpad, central executive, episodic buffer)
- Capacity limits (Miller's magical number 7 ± 2; Cowan's 4 ± 1)
- Encoding (acoustic, semantic, visual)
- Maintenance vs. manipulation
- Working memory and executive function

### Topic 5: Long-Term Memory I — Encoding and Retrieval (5 hours)
- Types of LTM (declarative vs. non-declarative; episodic vs. semantic)
- Levels-of-processing (Craik & Lockhart)
- Encoding specificity principle
- Retrieval cues and context
- Recall vs. recognition
- Serial position effect (primacy, recency)

### Topic 6: Long-Term Memory II — Forgetting and Distortion (5 hours)
- Ebbinghaus forgetting curve
- Decay vs. interference
- Retroactive vs. proactive interference
- Motivated forgetting (repression — controversial)
- Memory distortion (Loftus & Palmer, 1974)
- Misinformation effect
- False memories (Roediger & McDermott)

### Topic 7: Knowledge Representation (4 hours)
- Semantic networks (Collins & Quillian)
- Spreading activation
- Schemas and scripts
- Mental models
- Concepts and prototypes (Rosch)

### Topic 8: Language I — Production and Comprehension (4 hours)
- Language levels (phonemes, morphemes, syntax, semantics, pragmatics)
- Speech production (Levelt's model: conceptualization → formulation → articulation)
- Speech perception (categorical perception, McGurk effect)
- Word recognition (logogen model)
- Sentence processing (garden-path sentences)

### Topic 9: Language II — Bilingualism and Disorders (3 hours)
- Bilingualism (cognition effects: executive function boost)
- Language acquisition (nativist vs. learning theories)
- Critical period hypothesis
- Aphasias (Broca's, Wernicke's, conduction)

### Topic 10: Decision Making (5 hours)
- Dual-process theory (System 1 / System 2; Kahneman)
- Heuristics (availability, representativeness, anchoring)
- Biases (framing, sunk cost, confirmation)
- Prospect theory (loss aversion)
- Bounded rationality (Simon)
- Nudge theory (Thaler & Sunstein)

### Topic 11: Problem Solving and Creativity (3 hours)
- Problem space theory (Newell & Simon)
- Means-ends analysis
- Analogical reasoning
- Obstacles to problem solving (functional fixedness, mental set)
- Creativity (divergent vs. convergent thinking)

### Topic 12: Cognitive Neuroscience (3 hours)
- Localization of function
- Prefrontal cortex and executive function
- Hippocampus and memory
- Hemispheric specialization
- Neuroimaging methods (recap from PSYC7303)

## Key Concepts Glossary

| Term | Definition |
|---|---|
| Schema | Mental framework for organizing knowledge about a domain |
| Working memory | Active manipulation of information held in mind |
| Encoding | Converting information into a storable form |
| Retrieval | Accessing stored information |
| Heuristic | Mental shortcut that simplifies decisions (often at cost of accuracy) |
| Cognitive bias | Systematic deviation from normative rationality |
| Dual-process theory | Two modes of thinking: fast/automatic (S1) and slow/deliberate (S2) |
| Feature integration | Combining separate features (color, shape) into a unified object |
| Spreading activation | Memory retrieval via activation that spreads through connected nodes |
| Limited capacity | Resources (e.g., attention, working memory) that can handle only so much at once |
| Prospective memory | Remembering to perform a planned action in the future |
| Priming | Implicit memory effect: exposure influences subsequent processing |
| Levels of processing | Depth of semantic processing predicts retention (Craik & Lockhart) |

## Self-Check Questions

1. Explain the difference between bottom-up and top-down processing. Give an example of each.
2. Describe the cocktail party effect. What does it tell us about selective attention?
3. Outline Baddeley's model of working memory. What happens if one component is damaged?
4. What is the difference between retroactive and proactive interference? Construct an example of each.
5. Explain the misinformation effect. How might this apply to eyewitness testimony in legal contexts?
6. Describe Kahneman's System 1 and System 2. Give 3 examples of judgments that rely primarily on each.
7. What is the anchoring heuristic? How would you test for it experimentally?
8. Compare Loftus's findings on eyewitness memory with real-world forensic practice. What are the implications?
9. Define functional fixedness. How would you design an experiment to study it?
10. How do bilingual speakers differ from monolinguals in executive function? What neural mechanisms are proposed?

## Connection to Capstone

Cognitive psychology is **highly relevant** to your capstone if you choose a JDM (Judgment & Decision Making) or AI trust subfield:

- **System 1/System 2** is foundational to understanding decision styles
- **Heuristics and biases** (Kahneman) are core to behavioral finance research
- **Attention and working memory** relate to information overload in trading
- **Decision making under uncertainty** is a cognitive psych research domain
- **Memory distortion** connects to how traders recall past trades

**Strong fit topics:** "Personality and decision-making style in engineers" (uses heuristics, risk), "Stress and risk preferences in solo trading" (uses cognitive load theory)

## Sample Discussion Questions

- Cognitive psychology emerged as a reaction to behaviorism. What could not behaviorism explain that cognitive psychology could?
- A chess grandmaster can play "blindfold chess" against 10 opponents simultaneously. What does this tell us about expertise and working memory?
- Memory is reconstructive, not reproductive. What are the ethical implications for therapy, law, and history?
- A company wants to use nudges to influence employee behavior. Discuss the ethics.
- Compare the "rational actor" model of economics with the cognitive psychology view. Where do they diverge?
