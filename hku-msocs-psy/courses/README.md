## How to Use These Guides

Each guide follows the same structure:

1. **Course Info** — code, title, MSocSc comp status, BPsych equivalent, study month, hours
2. **Learning Objectives** — what you should be able to do after
3. **Topic Outline** — 10–14 topics for the course
4. **Required Resources** — textbook chapters, papers, videos
5. **Key Concepts Glossary** — terms you must know
6. **Self-Check Questions** — 8–10 questions to test your understanding
7. **Connection to Capstone** — how this course feeds your research project
8. **Sample Discussion Questions** — for AI / peer review

## Why Detailed Per-Course Files

- **Easier to study topic-by-topic** than scrolling through a single mapping table
- **Self-contained** — each course guide is independent
- **Git-trackable progress** — tick off topics as you complete them
- **AI-friendly** — paste a course file into AI for context-specific help
