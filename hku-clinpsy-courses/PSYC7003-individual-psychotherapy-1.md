# PSYC7003 — Individual Psychotherapy I

## Course Info

- **Code:** PSYC7003
- **Title:** Individual psychotherapy I
- **Program:** MSocSc in Clinical Psychology
- **Year:** 1
- **HKU Description:** Introduction to the theory and practice of individual psychotherapy. Major theoretical orientations (psychodynamic, CBT, humanistic, existential) and their application. Foundational therapeutic skills including rapport building, empathy, active listening, and case formulation.
- **Prerequisites:** PSYC7024
- **Assessment:** 100% coursework (typically skills practice, role-play, case formulation, recorded sessions review)
- **Notes:** Foundation psychotherapy course

## Learning Objectives

By the end of this course, you should be able to:

1. Describe major psychotherapy orientations
2. Demonstrate foundational therapeutic skills
3. Conduct initial case formulation
4. Apply empathy and active listening
5. Manage the therapeutic frame
6. Discuss evidence base for psychotherapy
7. Apply basic CBT and psychodynamic interventions

## Topic Outline

### Topic 1: Foundations of Psychotherapy (4 hours)
- History of psychotherapy
- Common factors
- Therapist effects
- Alliance and rupture-repair

### Topic 2: Psychodynamic Theory (6 hours)
- Freud and classical psychoanalysis
- Object relations
- Self psychology
- Attachment theory
- Defense mechanisms

### Topic 3: Cognitive-Behavioral Theory (6 hours)
- Beck's cognitive model
- Ellis's REBT
- Behavioral principles
- Cognitive distortions
- Behavioral experiments

### Topic 4: Humanistic and Existential (4 hours)
- Rogers and person-centered
- Gestalt therapy
- Existential therapy
- Meaning and authenticity

### Topic 5: Other Orientations (3 hours)
- IPT (Interpersonal)
- ACT (Acceptance and Commitment)
- DBT (Dialectical Behavior)
- Brief psychodynamic

### Topic 6: Therapeutic Skills (10 hours)
- Active listening
- Empathy
- Reflection
- Open/closed questions
- Silence
- Self-disclosure

### Topic 7: Case Formulation (8 hours)
- Presenting problem
- Predisposing, precipitating, perpetuating factors
- Cognitive case formulation (Beck)
- Plan formulation

### Topic 8: Beginning Phase Skills (6 hours)
- Rapport building
- Informed consent
- Treatment goals
- Expectations and structure

### Topic 9: Working Phase Skills (6 hours)
- Socratic questioning
- Behavioral activation
- Cognitive restructuring
- Insight-oriented interventions

### Topic 10: Ending Phase Skills (3 hours)
- Termination planning
- Relapse prevention
- Consolidation

## Key Concepts Glossary

| Term | Definition |
|---|---|
| Psychotherapy | Talk-based psychological treatment |
| Common factors | Elements common to all therapies |
| Therapeutic alliance | Collaborative bond between therapist and client |
| Transference | Client's feelings toward therapist from past relationships |
| Countertransference | Therapist's reactions to client |
| Cognitive distortion | Systematic error in thinking |
| Socratic questioning | Guided questioning to examine beliefs |
| Behavioral activation | Increasing engagement in reinforcing activities |
| Defense mechanism | Unconscious process protecting from anxiety |
| Working alliance | Agreement on goals and tasks |
| Empathy | Understanding client's experience |
| Active listening | Full attention to client communication |
