# PSYC8004 — Clinical Neuropsychology

## Course Info

- **Code:** PSYC8004
- **Title:** Clinical neuropsychology
- **Program:** MSocSc in Clinical Psychology
- **Year:** 2
- **HKU Description:** Clinical applications of neuropsychology. Neuroanatomy, neuropsychological assessment, differential diagnosis of neurological conditions, and rehabilitation. Builds on PSYC3054 / PSYC2111.
- **Prerequisites:** PSYC7018, PSYC7006
- **Assessment:** 100% coursework
- **Notes:** Clinical application of neuropsychology

## Learning Objectives

By the end of this course, you should be able to:

1. Apply neuroanatomy to clinical assessment
2. Conduct comprehensive neuropsychological evaluation
3. Differentially diagnose neurological conditions
4. Apply neuropsychological rehabilitation principles
5. Interpret neuroimaging findings
6. Communicate with medical professionals
7. Discuss ethical and cultural considerations

## Topic Outline

### Topic 1: Neuroanatomy Review (5 hours)
- Cortical and subcortical structures
- White matter tracts
- Vascular territories
- Functional networks

### Topic 2: Neurological Conditions (8 hours)
- Stroke
- Traumatic brain injury
- Brain tumors
- Epilepsy
- Multiple sclerosis
- Movement disorders
- Dementia

### Topic 3: Comprehensive Neuropsychological Battery (8 hours)
- Test selection
- Fixed vs. flexible battery
- Premorbid estimation
- Performance validity

### Topic 4: Domain-Specific Assessment (10 hours)
- Attention and processing speed
- Memory
- Language
- Visuospatial
- Executive function
- Mood and personality

### Topic 5: Differential Diagnosis (6 hours)
- Dementia vs. depression (pseudodementia)
- Mild cognitive impairment
- Alzheimer's vs. vascular
- FTD variants
- Lewy body

### Topic 6: Neuroimaging Integration (5 hours)
- CT/MRI interpretation
- Functional imaging (fMRI, FDG-PET)
- White matter changes
- Reporting

### Topic 7: Rehabilitation (5 hours)
- Cognitive rehabilitation
- Compensatory strategies
- Environmental modification
- Family education
- Return to work/school

### Topic 8: Pediatric Neuropsychology (3 hours)
- Developmental considerations
- Pediatric TBI
- Neurodevelopmental conditions
- School liaison

### Topic 9: Geriatric Neuropsychology (3 hours)
- Normal aging
- Mild NCD
- Dementia
- Capacity assessment

### Topic 10: Forensic and Capacity (3 hours)
- Testamentary capacity
- Fitness to plead
- Compensation evaluations
- Independent medical examinations

### Topic 11: Communication and Report Writing (4 hours)
- Medical team communication
- Comprehensive reports
- Feedback to patients/families

### Topic 12: Cultural and Ethical Issues (3 hours)
- Test bias in diverse populations
- HK Chinese considerations
- Consent and capacity issues
- End-of-life considerations

## Key Concepts Glossary

| Term | Definition |
|---|---|
| Neuropsychology | Brain-behavior relationships |
| NCD | Neurocognitive disorder |
| MCI | Mild cognitive impairment |
| FTD | Frontotemporal dementia |
| Pseudodementia | Cognitive impairment from depression |
| Premorbid | Functioning before illness |
| Performance validity | Effort testing |
| Aphasia | Language disorder |
| Apraxia | Motor planning disorder |
| Agnosia | Recognition disorder |
| TBI | Traumatic brain injury |
| Rehabilitation | Restoring function |
| Capacity | Decision-making ability |
