# PSYC7018 — Cognitive Assessment

## Course Info

- **Code:** PSYC7018
- **Title:** Cognitive assessment
- **Program:** MSocSc in Clinical Psychology
- **Year:** 1
- **HKU Description:** Theory and practice of cognitive assessment in clinical settings. Major intelligence tests (WAIS-IV, WISC-V, WPPSI-IV), memory tests (WMS-IV, RBANS), executive function tests, and integrated report writing.
- **Prerequisites:** PSYC7024
- **Assessment:** 100% coursework
- **Notes:** Combines test administration, scoring, interpretation, and report writing

## Learning Objectives

By the end of this course, you should be able to:

1. Administer, score, and interpret the WAIS-IV
2. Administer, score, and interpret the WISC-V
3. Administer memory tests
4. Assess executive functions
5. Integrate cognitive data for diagnosis
6. Write comprehensive cognitive assessment reports
7. Communicate findings to clients/families

## Topic Outline

### Topic 1: Foundations (4 hours)
- Psychometric principles
- Intelligence theories (Spearman, Cattell, Gardner, Sternberg)
- Cross-cultural assessment

### Topic 2: WAIS-IV (10 hours)
- Subtests and structure
- Administration and scoring
- Index scores (VCI, PRI, WMI, PSI)
- FSIQ interpretation
- Profile analysis
- Cultural considerations

### Topic 3: WISC-V and Children's Assessment (8 hours)
- WISC-V structure
- Administration with children
- Working with families
- School context

### Topic 4: Memory Assessment (8 hours)
- WMS-IV (Wechsler Memory Scale)
- RBANS (Repeatable Battery)
- CVLT (California Verbal Learning)
- BVMT-R (Brief Visuospatial Memory)

### Topic 5: Executive Function Assessment (6 hours)
- D-KEFS (Delis-Kaplan)
- Wisconsin Card Sorting Test
- Stroop
- Trail Making Test
- Behavioral observations

### Topic 6: Attention and Processing Speed (3 hours)
- CPT (Continuous Performance Test)
- Symbol Digit Modalities
- Reaction time

### Topic 7: Adaptive Functioning (3 hours)
- Vineland Adaptive Behavior Scales
- Adaptive Behavior Assessment System
- Cross-informant integration

### Topic 8: Differential Diagnosis (5 hours)
- Intellectual disability
- Specific learning disorder
- ADHD profile
- Cognitive decline
- Malingering and effort testing

### Topic 9: Report Writing (10 hours)
- Integrated cognitive reports
- Feedback sessions with clients/families
- Recommendations
- Communication with schools/employers

### Topic 10: Cultural and Ethical Issues (3 hours)
- Test bias
- Cultural validity
- Ethical use
- HK Chinese context

## Key Concepts Glossary

| Term | Definition |
|---|---|
| IQ | Intelligence Quotient |
| FSIQ | Full Scale IQ |
| VCI | Verbal Comprehension Index |
| PRI | Perceptual Reasoning Index |
| WMI | Working Memory Index |
| PSI | Processing Speed Index |
| Executive function | Higher-order cognitive control |
| Working memory | Active manipulation of info |
| Processing speed | Speed of cognitive operations |
| Adaptive behavior | Everyday functioning |
| Effort testing | Assessment of validity |
| Malingering | Faking for external gain |
