# PSYC7004 — Individual Psychotherapy II

## Course Info

- **Code:** PSYC7004
- **Title:** Individual psychotherapy II
- **Program:** MSocSc in Clinical Psychology
- **Year:** 1
- **HKU Description:** Continuation of Individual Psychotherapy I. Advanced skills in CBT and psychodynamic therapy, integration of techniques, application to specific disorders, and management of common therapeutic challenges (resistance, dropout, ruptures).
- **Prerequisites:** PSYC7003
- **Assessment:** 100% coursework
- **Notes:** Builds on PSYC7003

## Learning Objectives

By the end of this course, you should be able to:

1. Apply advanced CBT techniques
2. Apply advanced psychodynamic interventions
3. Integrate techniques across orientations
4. Address resistance and ruptures
5. Manage dropout risk
6. Adapt psychotherapy to specific disorders
7. Evaluate therapeutic outcomes

## Key Concepts Glossary

| Term | Definition |
|---|---|
| Schema | Core belief structure |
| Behavioral experiment | Test of belief using behavioral test |
| Exposure | Confronting feared stimulus |
| Object relations | Internal representations of self/other |
| Transference-focused | Therapy based on transference work |
| Mentalization | Understanding mental states |
| Rupture | Breakdown in therapeutic alliance |
| Repair | Restoring alliance |
| Resistance | Client's avoidance of change |
| Therapeutic alliance | Collaborative bond |
| Integrative | Combining multiple approaches |

## Self-Check Questions

1. Compare schema-focused and traditional CBT.
2. How do you recognize and repair therapeutic ruptures?
3. Describe mentalization-based treatment.
4. Discuss dropout prevention strategies.
5. Design an outcome monitoring plan for a case.
