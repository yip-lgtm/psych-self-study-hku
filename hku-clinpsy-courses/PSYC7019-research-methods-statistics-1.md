# PSYC7019 — Research Methods and Statistics I

## Course Info

- **Code:** PSYC7019
- **Title:** Research methods and statistics I
- **Program:** MSocSc in Clinical Psychology
- **Year:** 1
- **HKU Description:** Quantitative research methods and statistics for clinical psychology. Research design, descriptive statistics, hypothesis testing, ANOVA, regression, and introduction to multivariate methods.
- **Prerequisites:** PSYC7024
- **Assessment:** 100% coursework (typically data analysis assignments, research design exercises, exam)
- **Notes:** Foundation quantitative methods; supports dissertation

## Learning Objectives

By the end of this course, you should be able to:

1. Design quantitative research studies
2. Conduct power analysis
3. Apply descriptive and inferential statistics
4. Perform ANOVA and regression
5. Use SPSS, R, or Python for analysis
6. Interpret statistical results
7. Critically evaluate quantitative research

## Topic Outline

### Topic 1: Research Design (5 hours)
- Experimental and quasi-experimental designs
- Cross-sectional vs. longitudinal
- Sampling methods
- Threats to validity

### Topic 2: Measurement (4 hours)
- Reliability and validity
- Factor analysis introduction
- Scale development basics

### Topic 3: Descriptive Statistics (4 hours)
- Measures of central tendency and dispersion
- Distributions
- Z-scores
- Visualization

### Topic 4: Hypothesis Testing (6 hours)
- Null and alternative hypotheses
- Type I and II errors
- Power
- Parametric and non-parametric tests
- Effect sizes

### Topic 5: t-Tests (4 hours)
- One-sample, independent, paired
- Assumptions and violations
- Effect size (Cohen's d)
- Reporting

### Topic 6: ANOVA (8 hours)
- One-way ANOVA
- Factorial ANOVA
- Repeated-measures ANOVA
- Post-hoc tests
- Effect size (η²)

### Topic 7: Correlation and Regression (8 hours)
- Pearson and Spearman correlation
- Simple linear regression
- Multiple regression
- Assumptions and diagnostics

### Topic 8: Chi-Square (3 hours)
- Goodness-of-fit
- Test of independence
- Effect size (Cramér's V)

### Topic 9: Non-Parametric Tests (3 hours)
- Mann-Whitney U
- Wilcoxon signed-rank
- Kruskal-Wallis

### Topic 10: Software (8 hours)
- SPSS, R, or Python
- Data import/cleaning
- Common analyses
- Output interpretation

### Topic 11: Reporting and APA Style (4 hours)
- Results section
- Tables and figures
- APA 7th statistical reporting

## Key Concepts Glossary

| Term | Definition |
|---|---|
| Power | Probability of detecting true effect |
| Effect size | Standardized magnitude |
| ANOVA | Analysis of variance |
| eta-squared | Variance explained (η²) |
| Cohen's d | Standardized mean difference |
| Type I error | False positive |
| Type II error | False negative |
| Null hypothesis | No effect |
| Parametric | Assumes normal distribution |
| Non-parametric | Distribution-free |
| Multiple regression | Multiple predictors |
| Assumptions | Conditions for valid inference |
