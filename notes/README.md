# Daily Study Notes

> Free-form notes. One folder per month.

```
notes/
├── 2026-06/    ← Month 1
├── 2026-07/    ← Month 2
├── ...
└── 2026-12/    ← Month 6 + application prep
```

## Naming Convention

```
YYYY-MM-DD-topic.md
```

Example: `2026-06-21-jasp-setup.md`

## What Goes Here

- Lecture notes
- Lightbulb moments
- Confusion + how you resolved it
- Connections to other things
- Random ideas for capstone
- Things to ask AI / peers

## What Does NOT Go Here

- Full paper logs (those go in `papers/classic-readings.md`)
- Month milestone tracking (that's in `progress/monthly-milestones.md`)
- Streak log (that's in `progress/streak.md`)
- Capstone work (that's in `capstone/`)
