## Week 14 — Subfield Deep Dive (Block 1)

### Choose based on your Month 3 commitment. Sample subfields:

### A. Judgment & Decision Making
- Read: Kahneman "Thinking, Fast and Slow" Ch 1–10
- Read: 3 papers from *Judgment and Decision Making* journal
- Key concepts: heuristics, biases, prospect theory, dual-process

### B. Health Psychology
- Read: Taylor "Health Psychology" Ch 1–5
- Read: 3 papers on self-regulation / health behavior change
- Key concepts: Theory of Planned Behavior, Transtheoretical Model

### C. Quantitative / Psychometrics
- Read: Furr "Psychometrics" Ch 1–4
- Run: confirmatory factor analysis in R (`lavaan`) on a public dataset
- Key concepts: reliability, validity, factor analysis, IRT

### D. Educational Psychology
- Read: Ormrod "Educational Psychology" Ch 1–5
- Read: 3 papers on metacognition / self-regulated learning
- Key concepts: zone of proximal development, cognitive load, mastery vs performance goals

### E. Cognitive Neuroscience / Human-AI Interaction
- Read: Gazzaniga "Cognitive Neuroscience" Ch 1–3
- Read: 3 papers on decision-making neuroscience (e.g., fMRI studies of value-based choice)
- Key concepts: prefrontal cortex, valuation systems, prediction error

### Deliverable (all tracks)
- 1-page summary of the subfield's main methods, debates, and current frontier
