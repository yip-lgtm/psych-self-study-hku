# PSYC7003 — Individual Psychotherapy I (PhD)

## Course Info (from HKU Catalog)

- **Code:** PSYC7003
- **Title:** Individual psychotherapy I
- **Program:** PhD Clinical Psychology (4-year)
- **Year:** 1
- **HKU Description:** Doctoral-level introduction to individual psychotherapy. Major theoretical orientations, evidence base, and clinical application. Emphasizes research-practice integration and outcome monitoring.
- **Prerequisites:** PSYC7024
- **Assessment:** 100% coursework
- **Notes:** Doctoral-level psychotherapy foundation

## Learning Objectives (PhD-level)

By the end of this course, you should be able to:

1. Describe major psychotherapy orientations with sophistication
2. Critically evaluate evidence base
3. Demonstrate doctoral-level therapeutic skills
4. Conduct research-informed case formulation
5. Manage therapeutic frame
6. Discuss common factors and specific techniques
7. Apply outcome monitoring

## Topic Outline (PhD-level)

### Topic 1: Foundations of Psychotherapy (4 hours)
- Doctoral-level history
- Common factors
- Therapist effects
- Alliance rupture-repair
- Dodo bird verdict

### Topic 2: Psychodynamic Theory (6 hours)
- Doctoral-level psychodynamic theory
- Object relations
- Self psychology
- Attachment theory
- Defense mechanisms

### Topic 3: Cognitive-Behavioral Theory (6 hours)
- Beck and Ellis
- Third-wave CBT
- Behavioral principles
- Mechanisms of change

### Topic 4: Humanistic and Existential (4 hours)
- Rogers
- Gestalt
- Existential
- Meaning

### Topic 5: Other Orientations (3 hours)
- IPT, ACT, DBT
- EFT
- Brief psychodynamic

### Topic 6: Therapeutic Skills (10 hours)
- Doctoral-level active listening
- Empathy at doctoral level
- Reflection
- Open/closed questions
- Silence
- Self-disclosure

### Topic 7: Case Formulation (8 hours)
- Doctoral-level formulation
- Beck's cognitive case formulation
- Plan formulation
- Research-informed

### Topic 8: Beginning Phase Skills (6 hours)
- Rapport building
- Informed consent
- Treatment goals
- Expectations and structure

### Topic 9: Working Phase Skills (6 hours)
- Doctoral-level Socratic questioning
- Behavioral activation
- Cognitive restructuring
- Insight-oriented interventions

### Topic 10: Ending Phase Skills (3 hours)
- Termination planning
- Relapse prevention
- Consolidation

## Key Concepts Glossary

| Term | Definition |
|---|---|
| Common factors | Elements common to all therapies |
| Dodo bird verdict | All therapies equally effective |
| Therapeutic alliance | Collaborative bond |
| Transference | Client's feelings from past |
| Countertransference | Therapist's reactions |
| Cognitive distortion | Systematic thinking error |
| Socratic questioning | Guided questioning |
| Behavioral activation | Increasing engagement |
| Defense mechanism | Unconscious protective process |
| Empathy | Understanding client's experience |
| Active listening | Full attention |
| Outcome monitoring | Tracking progress |
