# PSYC7018 — Cognitive Assessment (PhD)

## Course Info (from HKU Catalog)

- **Code:** PSYC7018
- **Title:** Cognitive assessment
- **Program:** PhD Clinical Psychology (4-year)
- **Year:** 1
- **HKU Description:** Doctoral-level theory and practice of cognitive assessment. Major intelligence tests, memory tests, executive function tests, advanced integrative report writing. Research-informed approach.
- **Prerequisites:** PSYC7024
- **Assessment:** 100% coursework
- **Notes:** Doctoral-level cognitive assessment

## Learning Objectives (PhD-level)

By the end of this course, you should be able to:

1. Apply doctoral-level cognitive assessment
2. Administer, score, and interpret WAIS-IV with sophistication
3. Administer, score, and interpret WISC-V
4. Administer memory tests at doctoral level
5. Assess executive functions with research literacy
6. Integrate cognitive data for doctoral-level diagnosis
7. Write doctoral-level cognitive assessment reports
8. Apply assessment in diverse populations with research literacy

## Topic Outline (PhD-level)

### Topic 1: Foundations (4 hours)
- Doctoral-level psychometrics
- Intelligence theories advanced
- Cross-cultural assessment
- Cultural validity research

### Topic 2: WAIS-IV (10 hours)
- Subtests and structure
- Administration and scoring
- Index scores
- FSIQ interpretation
- Profile analysis
- Research on WAIS-IV
- Cultural considerations

### Topic 3: WISC-V and Children's Assessment (8 hours)
- WISC-V structure
- Administration with children
- Family work
- School context
- Research on WISC-V

### Topic 4: Memory Assessment (8 hours)
- WMS-IV
- RBANS
- CVLT
- BVMT-R
- Research on memory tests

### Topic 5: Executive Function Assessment (6 hours)
- D-KEFS
- WCST
- Stroop
- Trail Making
- Behavioral observations
- Research on EF tests

### Topic 6: Attention and Processing Speed (3 hours)
- CPT
- Symbol Digit Modalities
- Reaction time
- Research

### Topic 7: Adaptive Functioning (3 hours)
- Vineland
- ABAS
- Cross-informant integration

### Topic 8: Doctoral-Level Differential Diagnosis (5 hours)
- Intellectual disability
- Specific learning disorder
- ADHD profile
- Cognitive decline
- Malingering and effort testing
- Research on differential diagnosis

### Topic 9: Doctoral-Level Report Writing (10 hours)
- Integrated cognitive reports
- Doctoral-level synthesis
- Research integration
- Feedback sessions
- Communication with schools/employers

### Topic 10: Cultural and Ethical Issues (3 hours)
- Doctoral-level test bias research
- Cultural validity
- Ethical use
- HK Chinese context
