# Psychology Self-Study (HKU Conversion Prep)

> **Mission:** Build a HKU MSocSc(Psy) conversion-ready foundation in 6 months while working full-time as a structural engineer. Target: Quantitative / Health / Decision-Making Psychology subfields.

**Study start:** June 2026
**Conversion target:** HKU MSocSc(Psy) 2027 intake
**Weekly load:** 15–25 hours (split: 2h × 5 weekday evenings + 5h weekend)

## 🎯 Program Decision Matrix

| Criterion | MSocSc(Psy) | BPsych (4-yr) | MSocSc Clinical | PhD Clinical |
|---|---|---|---|---|
| **Duration** | 1 yr FT | 4 yrs FT | 2 yrs FT | 4 yrs FT |
| **Tuition 2026-27** | ~HK$209,800 | (UG funded) | ~HK$209,800 (FT) | ~HK$42,000/yr (PGS) |
| **Focus** | Research methods | Broad foundations | Practitioner training | Scientist-practitioner |
| **Career path** | PhD / RA / consulting | Clinical / I-O / research | Clinical practice | Academic / research |
| **Research** | Dissertation (required) | Honours thesis (4th yr) | Dissertation + PSU | Original doctoral thesis |
| **Practicum** | None | Field placement (1) | 5 practicums (extensive) | 5 practicums + PSU |
| **Subfield fit (JDM/Health/Quant)** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ |
| **Compatible with current job** | ⭐⭐⭐⭐ | ❌ (full-time) | ⭐ | ❌ |
| **Application deadline** | Late Jan/early Feb 2027 | (UG only) | Variable | Variable |
| **Target** | **PRIMARY** | Back-up | Stretch | Stretch (long-term) |

## How to Use This Repo

1. **Daily** → open `progress/streak.md`, mark today's session
2. **Weekly** → fill in `progress/weekly-review-template.md` (every Sunday)
3. **Monthly** → run `progress/monthly-milestones.md` self-check, archive to `notes/YYYY-MM/`
4. **Per-paper** → log in `papers/classic-readings.md` (3-pass method)
5. **Capstone** → everything goes in `capstone/`; aim for 4000–6000 words by Month 6
6. **For MSocSc(Psy) application** → start with `hku-msocs-psy/courses/README.md` for the 8 compulsory courses
7. **For reference** → use `hku-bpsych-courses/` to see what's covered in a 4-year undergrad program
8. **For stretch goals** → browse `hku-clinpsy-courses/` or `hku-phd-clinpsy-courses/`
